Lista de Definicion

Metalenguajes

SML
Metalenguaje para la definicion de otros lenguajes de marcado
XML
Lenguaje basado en SGML y que se emlea para describir datos
RSS
RSS son las siglas de Really Simples Syndication, un formato XML para indicar o compartir contenido en la web
GML
Lenguaje de Marcado Geografico al modelaje, transporte y almacenamiento
XUL
Lenguaje derivados de XML para determinadas aplicaciones
