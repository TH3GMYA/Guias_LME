Estilos embebidos en las etiquetas html

Usar estilos embebidos en las etiquetas HTML en el peor y el menos utilizado de los metodos para agregar reglas de estilos. No es nada recomendable por la dificultad que implica realizar un cambio posterior1

Esta forma de incluir css directamente en el elemento HTML solo se utiliza en determinadas situacionesen las que se debe incluir estilos muy especificos para un solo elemento en el concreto

Para insertar un estilo CSS en un elemento se utiliza la propiedad Style de HTML, que en ningun momento se debe confundir con la etiqueta <style>que se introducen en la cabeza del documento.
